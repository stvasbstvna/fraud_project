Fraud Project
Project overview

This project focuses on banking smishing detection and prevention. My goal is to study whether SMS content, scam tactics, and URL features can improve the detection of banking smishing, and to use this research as a foundation for a real product or platform that can help people avoid scams.

Current focus

Right now I am in the early stage of the research process. I am reviewing papers, organizing datasets, planning experiments, and setting up the workflow for GitHub and Overleaf.

Main datasets
Smishing-4C
SmishTank
PhiUSIIL
Main models
Random Forest
Logistic Regression
Research goal

The main goal is to build a strong research pipeline first, then use it to support a practical safety tool or platform related to banking smishing.
